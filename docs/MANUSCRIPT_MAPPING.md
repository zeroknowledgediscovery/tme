## Symbolic GPT/human values

The values `H_human = 0.77` and `H_gpt = 0.74` bits/character are imported from NERO and stored in:

```text
inputs/symbolic_entropy_inputs.json
```

The TME repository computes only the manuscript-level transformation:

```text
log2(q_GPT/q_H) = n_star (H_GPT - H_H) + rho_n_star
I_H + 1 = q_H N_V/N_G
I_GPT + 1 = (I_H + 1) 2^{n_star(H_GPT-H_H)+rho_n_star}
```

The entropy-rate estimation workflow itself should be cited to the NERO repository.
